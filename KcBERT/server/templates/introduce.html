Introducing an AI system which detects online hatred comments and analyzes community’s sentiments using KcBERT

Overview

KcBERT model is a BERT-based model pretrained from scratch using Korean news comments data

  • 3000 vocabulary size
  • 100 million parameters
  • 435MB model size
  • Fine-tuned on Unsmile dataset

Performance

KcBERT-based hatred expression classifier outperforms KoBERT-based classifier w.r.t predicting class labels as well as to converging to minimum
